# Camotes Runner - AI Development Rules

## Project Overview

Camotes Runner is a mobile application for Camotes Island that connects customers with local motorcycle riders for transportation, errands, grocery pickup, medicine pickup, document delivery, and tourism transportation.

The application is being built using:

- React Native
- Expo
- Expo Router
- TypeScript
- Mobile-first design
- Play Store deployment target

## Development Rules

- Never remove existing working features.
- Preserve current navigation structure.
- Explain all changes before and after implementation.
- Use reusable components whenever possible.
- Keep code beginner-friendly.
- Use TypeScript.
- Avoid unnecessary complexity.
- Maintain clean folder organization.
- Mobile-first responsive design.
- Follow Expo Router conventions.
